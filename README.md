## 🎯 Tính năng chính

- Quản lý sản phẩm theo thương hiệu địa phương
- Hỗ trợ biến thể sản phẩm (size, màu sắc)
- Quản lý giỏ hàng và đơn hàng
- Tích hợp thanh toán đa dạng (COD, MoMo, VNPay)
- Hệ thống đánh giá và khuyến mãi
